enrollmentsplugin plugin for Tutor


Installation
************

.. code-block:: bash

    pip install git+https://github.com/rehmansheikh222/tutor-contrib-enrollmentsplugin

Usage
*****

.. code-block:: bash

    tutor plugins enable enrollmentsplugin
